❓ שאלות נפוצות לפני שמתחילים

אני לא מתכנת/ת — זה בשבילי? ▼
בהחלט. אפליקציית Claude Desktop מתאימה לכולם — יש בה ממשק ויזואלי ולא צריך לדעת להשתמש בטרמינל. מורידים, מתקינים, ועובדים.
מה ההבדל בין claude.ai (האתר) לאפליקציית Desktop? ▼
האתר (claude.ai) הוא chat רגיל — כותבים שאלות, מקבלים תשובות. האפליקציה מוסיפה מעל זה Tab מיוחד בשם "Code" שמאפשר ל-Claude לגשת ולערוך את הקבצים האמיתיים שלכם על המחשב. זה ממש אחר.
צריך לשלם? ▼
האפליקציה עצמה חינמית להורדה. כדי להשתמש ב-Code Tab (שזה הלב של Claude Code) צריך מנוי Claude Pro — כ-20 דולר לחודש. יש גרסת Chat חינמית, אבל ללא גישה לקבצים.
האם זה בטוח — Claude יגש לכל הקבצים שלי? ▼
Claude ניגש רק לתיקייה שבחרתם בעצמכם לתת לו גישה. הוא לא יכול לגלוש לשאר המחשב. כל פעולה שהוא עושה מוצגת לכם ואפשר לאשר או לבטל.
Windows או Mac — יש הבדל? ▼
האפליקציה עובדת על שניהם. ב-Windows יש דרישה אחת נוספת — להתקין Git לפני שפותחים את Code Tab לראשונה (מוסבר בשלב 3 למטה).

🚀 שלב 1 — הורידו את אפליקציית Claude Desktop

הורידו את האפליקציה מהאתר הרשמי

ההורדה היא מ-Anthropic ישירות — לא מחנויות אפליקציות של צד שלישי. הקליקו על הכפתור המתאים למחשב שלכם:

עבור לדף ההורדה הרשמי ← claude.com/download

בדף ההורדה תמצאו כפתורים נפרדים ל-Windows וגם ל-Mac. הורידו את הקובץ המתאים למחשב שלכם והפעילו אותו.

א

Windows — הפעלת קובץ ההתקנה

לאחר ההורדה תקבלו קובץ בשם משהו כמו Claude-Setup.exe. לחצו עליו פעמיים, אשרו את בקשת המנהל שתופיע, וחכו שיסתיים.

אחרי ההתקנה, האפליקציה תופיע בתפריט Start.
ב

Mac — פתיחת קובץ ה-DMG

לאחר ההורדה תקבלו קובץ .dmg. פתחו אותו, גררו את האפליקציה לתיקיית Applications, וסגרו.

פתחו את Claude מתיקיית Applications או מ-Spotlight (Cmd+Space).

🔑 שלב 2 — התחברות לחשבון

1

פתחו את האפליקציה והתחברו

האפליקציה תבקש להתחבר עם חשבון Anthropic. אם אין לכם חשבון — אפשר ליצור אחד בחינם באתר claude.ai.

2

וודאו שיש לכם מנוי Pro

כדי להשתמש ב-Code Tab (שהוא הלב של Claude Code) נדרש מנוי Claude Pro (~$20/חודש). אם תלחצו על ה-Code Tab ותראו הודעת שדרוג — זה הסימן.

לניהול המנוי: היכנסו ל-claude.ai ← הגדרות ← Plans

🪟 שלב 3 — Windows בלבד: התקנת Git

ב-Mac זה לא נדרש. ב-Windows חייבים להתקין Git לפני שפותחים את Code Tab לראשונה — בלי זה הוא לא יעבוד.

1

הורידו Git for Windows

היכנסו לאתר הרשמי: git-scm.com/downloads/win ולחצו על "Download for Windows".

2

הריצו את המתקין

פתחו את קובץ ה-exe שהורדתם. אפשר ללחוץ "Next" בכל המסכים — ברירות המחדל בסדר גמור.

לא לדאוג מהכמות הגדולה של אפשרויות — ברירות המחדל מתאימות לרוב הצרכים.
3

הפעילו מחדש את אפליקציית Claude

לאחר התקנת Git, סגרו ופתחו שוב את אפליקציית Claude Desktop. עכשיו Code Tab יעבוד.

💻 שלב 4 — עבודה עם Code Tab

בתוך האפליקציה תראו בחלק העליון שלושה Tabs: Chat, Cowork, ו-Code. לחצו על Code.

1

לחצו על "Code" בחלק העליון

ה-Code Tab הוא המקום שבו Claude Code עובד. זה מה שמאפשר לו לגשת לקבצים שלכם ולערוך אותם.

2

בחרו "Local" ואחר כך "Select folder"

תופיע שאלה על סביבת עבודה — בחרו Local (זה עובד על הקבצים שלכם במחשב). אחר כך לחצו "Select folder" ובחרו את התיקייה שרוצים לעבוד עליה.

תיקייה? זה פשוט — יצרתם תיקייה חדשה בשם "הפרויקט שלי"? זה מה שבוחרים. Claude יוכל לראות ולערוך רק קבצים בתוכה.
3

בחרו מודל ותתחילו

יש תפריט לבחירת מודל — ברירת המחדל מתאימה. עכשיו פשוט כתבו בעברית מה אתם רוצים שקלוד יעשה, ולחצו Enter.

לדוגמה: "צור לי קובץ HTML עם דף בית פשוט בעברית" — וקלוד יכתוב את הקובץ ישירות בתיקייה שבחרתם.
4

ראו את השינויים לפני שמאשרים

Claude מציג לכם תצוגת Diff — מה שינה, מה הוסיף, מה מחק. ירוק = נוסף, אדום = הוסר. אפשר לאשר הכל, לאשר רק חלק, או לבקש שינויים.

📦 רוצים גם את הכלים בטרמינל? (אופציונלי)

אם תרצו להשתמש ב-Claude Code גם דרך הטרמינל בעתיד (למתקדמים), סקריפט זה מתקין בפקודה אחת את Git, Node.js, ו-Claude Code CLI:

פתחו PowerShell כמנהל (לחצן ימני → "הפעל כמנהל") והדביקו:

irm "https://raw.githubusercontent.com/RanNahmany/claude-code-one-click-install/main/windows/install.bat" -OutFile install.bat; .\install.bat
PowerShell כמנהל: חיפוש ← PowerShell ← לחצן ימני ← "הפעל כמנהל". בסיום — פתחו חלון PowerShell חדש.

פתחו Terminal (Cmd+Space ← Terminal) והדביקו:

curl -fsSL https://raw.githubusercontent.com/RanNahmany/claude-code-one-click-install/main/mac/install.sh | bash
אם מבקשים סיסמה: הקלידו את סיסמת ה-Mac. הסיסמה לא מוצגת על המסך — זה נורמלי. לחצו Enter.

הסקריפט מוצע כאן כאופציה בלבד. לרוב המשתמשים — אפליקציית Claude Desktop מספיקה לחלוטין ואין צורך בסקריפט.

🔧 בעיות נפוצות ואיך פותרים

Code Tab מראה הודעת "Upgrade"

הסיבה: לא מחובר למנוי Pro.
פתרון: היכנסו ל-claude.ai ← הגדרות ← Plans ושדרגו לPro.

Windows: Code Tab לא נפתח / שגיאה

הסיבה: Git לא מותקן.
פתרון: התקינו Git from git-scm.com/downloads/win, ואחר כך הפעילו מחדש את אפליקציית Claude.

שגיאת 403 / "Sign in again"

הסיבה: תוקף ההתחברות פג.
פתרון: התנתקו מהאפליקציה והתחברו שוב.

האפליקציה לא מוצאת את התיקייה שבחרתי

נסו לבחור שוב עם "Select folder". ודאו שהתיקייה קיימת ואין בנתיב שלה תווים מיוחדים.

⚙️ בעיות עם הסקריפט (למי שהריץ אותו)

אם הסקריפט נכשל, אפשר להריץ במצב Debug לראות את הפירוט:

Windows (PowerShell כמנהל):

.\install.bat --debug

Mac (Terminal):

curl -fsSL https://raw.githubusercontent.com/RanNahmany/claude-code-one-click-install/main/mac/install.sh | bash -s -- --debug
